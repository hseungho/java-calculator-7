## 기능 명세서

### 기능 목록
| 기능ID |         기능명         |                               설명                                |
|:----:|:-------------------:|:---------------------------------------------------------------:|
|  1   |   콘솔에 텍스트를 출력         |                      콘솔에 특정 텍스트(들)를 출력한다.                       |
|  2   |     콘솔에 텍스트를 입력     |                     사용자가 콘솔을 통해 텍스트를 입력한다.                      |
|  3   | 입력 문자열을 기본 구분자로 분리  |          사용자가 입력한 문자열에서 기본 구분자 (`,`, `:`)로 문자열을 분리한다.           |
|  4   | 입력 문자열을 커스텀 구분자로 분리 | 사용자가 입력한 문자열 앞부분의 `//`와 `\n` 사이에 위치한 문자를 커스텀 구분자로 하여 문자열을 구분한다. |
|  5   | 구분된 숫자들 합산 |                     구분자로 분리한 문자들을 모두 합산한다.                      |
